# Premium Outfit Hub

Проект создан для заказа с [YouDo](https://youdo.com).

---

## Формулировка задачи

**Создать интернет-магазин под ключ**

Нужно сделать интернет-магазин одежды

### Задачи:

- Настроить дизайн и каталог
- Подключить онлайн-оплату и доставку (СДЭК)
- Привязать домен

В отклике присылайте примеры ваших интернет-магазинов, цену и сроки.

---

## Технологии

- React 19 + TypeScript
- TanStack Router (клиентский роутинг)
- Tailwind CSS 4 + shadcn/ui
- Vite 8
- Деплой: GitHub Pages
